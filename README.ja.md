## 一言でいうと

Domain Roulette Launch Labは、name.comのルーレットで出た3つのドメインを、提出できる事業案に変えるための道具です。

## 誰の何を助けるか

対象は、DeveloperWeek NYのname.com Domain Rouletteに参加するチームです。

課題は、ランダムに出たドメインを面白いだけで終わらせず、審査員に伝わる事業案にすることです。

このアプリは、3つのドメインを比較し、勝ち候補、MVP、弱点、提出コピーをまとめます。

## はじめての使い方

1. 公開アプリを開く
2. 3つのドメインを入れる
3. `分析する` を押す
4. 勝ち候補、MVP、提出コピーを見る

## 30秒レビュー手順

1. 3つのドメインを見る
2. どのドメインが一番強いかを見る
3. MVPと提出コピーを見る

## name.comトラックに刺さる理由

このトラックで大事なのは、面白いドメイン名を引くだけではありません。
その名前から、ちゃんとしたプロダクトを作れるかです。

このアプリは、次の5点を画面で見せます。

- 創造的な解釈: ドメイン名を、プロダクトの制約として使う
- 技術実装: 点数、比較、MVP、提出文を一画面で出す
- プロダクトの磨き: 日本語/英語、30秒レビュー、主張の境界を最初から見せる
- コンセプトの強さ: 1人のユーザー、1つの困りごと、1つの行動に絞る
- ドメインとの接続: ドメイン名が、ユーザー、機能、提出ストーリーを決める

## 画面の見方

- `ドメイン比較`: 覚えやすさ、意味の明確さ、収益性、実装しやすさを見る
- `事業案`: 誰の課題をどう解くかを見る
- `MVP`: 最初の3日で何を作るかを見る
- `提出コピー`: Devpostに使える説明文を見る

## 主張の境界

このアプリは、name.comの公式ルーレットそのものではありません。

公式に割り当てられた3つのドメインを入れたあと、チームが素早く判断するための道具です。
スコアは作業仮説であり、客観的な事業価値を保証するものではありません。

最終提出前に、サンプルの3ドメインを公式に割り当てられた3ドメインへ差し替えます。
